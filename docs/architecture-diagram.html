Harikatha Live Agent

Real-time AI-Powered Sanskrit Recitation with Gemini Live API

Gemini Live Agent Challenge 2026 | Hackathon Submission

Gemini Live API Semantic Search Vector Database Cloud Run WebSocket Streaming

System Features

🎤 Real-time Speech Processing

WebSocket-based streaming integration with Gemini Live API for instantaneous speech-to-text conversion and intent understanding without perceptible latency.

🔍 Semantic Search

Vertex AI embeddings power intelligent vector search across Harikatha texts, enabling contextual retrieval of relevant Sanskrit recitations and spiritual wisdom.

⚡ Low-Latency Architecture

Google Cloud Run's serverless platform ensures rapid response times with automatic scaling, optimized for real-time conversational AI experiences.

🎬 Multi-modal Output

Seamlessly combine generated responses with authentic audio and video playback of Gurudeva's actual Harikatha recitations, timestamps, and transcripts.

🔧 Function Calling

Gemini Live API's native function calling enables the search_harikatha tool to integrate directly into the conversational flow for intelligent tool use.

🌍 Scalable Data Backend

Firestore's globally distributed vector database handles millions of embeddings, ensuring fast retrieval and consistent performance worldwide.

Technology Stack

Frontend: React/TypeScript with WebSocket client
Backend: FastAPI (Python) on Google Cloud Run
AI/ML: Google Gemini Live API
Embeddings: Vertex AI Text Embeddings
Database: Firestore Vector Search
Audio/Video: Cloud Storage + Streaming
Infrastructure: Google Cloud Platform
Protocol: WebSocket for real-time streaming
